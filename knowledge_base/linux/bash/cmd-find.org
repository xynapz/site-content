* The /find/ command
The /find/ command[fn:1] is used to search for files in a directory hierarchy. The general synposis of the find command is */find [options] [starting-point] [expressions]/*.

** Options
Options control the behaviour of the find command, like treatment of symbolic links. The five real 'options' *-H*, *-L*, *-P*, *-D* and *-O* must appear before the first path name, if they are being used at all. The *-H*, *-L*, *-P* options control the treatment of symbolic links

[fn:1] See the [[https://man7.org/linux/man-pages/man1/find.1.html][find(1) man page]] for full documentation.
